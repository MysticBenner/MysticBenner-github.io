Machine Learning algorithms

  • supervised learning

    • regression problem (回归问题，连续的变量)
    • classification problem（分类问题，有限个值的离散变量）

  • Unsupervised learning

    • clustering problem(聚类问题)

  • difference of two types of learning

    ​ 看结果是否是有明确的输出。如果结果有明显的输出判定，那就是监督学习。但是如果输出结果完全没有预期，那就是无监督学习。
