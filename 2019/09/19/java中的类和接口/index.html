java中的类和接口

发表于 2019-09-19 阅读次数：

  • java中的继承

    关键字： extend

    声明：类名 对象名 = new 类名（）

  • 运行时多态的实现

    • 继承
    • 方法覆盖（override）
    • 父类对象指向子类对象

    static关键字

  • 在java中相当于伪全局变量

  • 在类被加载时static块也会被加载

  • static能够在没有创建对象的条件下被类直接调用

  • static块只加载一次，能优化程序结构以及提高程序效率

  • java中的四种访问权限

  • 抽象类

关键字：abstract/extend

将一类相同事物的共同特征抽象为抽象类，不纠结于具体的方法实现。

  • 接口

关键字：interface/implements

抽象类主要是用来抽象类别，接口主要是用来抽象方法功能。当你关注事物的本质的时候，请用抽象类；当你关注一种操作的时候，用接口。

  • 内部类

用于访问外部类中的一些方法或者变量，或者用于使外部类可以唯一调用内部类中的方法。
